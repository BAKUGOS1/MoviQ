# MoviQ PRD v3 — Final Reference

## Product

MoviQ is an AI-native video editing and short-form clip intelligence platform.

## Core Loop

```text
Upload -> Transcript -> Ranked Clips -> Web Editor -> Validated MP4 Export
```

## MVP Goals

- Convert long-form video into short clips.
- Preserve human editorial control.
- Provide a premium browser editor.
- Validate final MP4 exports server-side.
- Keep the workflow compliant and audit-friendly.

## MVP Features

1. Authentication and workspace selection
2. Project creation
3. Video upload
4. Media processing status
5. Transcript review
6. AI-ranked clip candidates
7. Clip candidate explanation and confidence
8. Manual clip selection and override
9. Web editor shell
10. Timeline, caption and style controls
11. Render job creation
12. Validated export artifact
13. Export history

## Non-MVP

- Marketplace
- Enterprise SSO
- Mobile editor parity
- Full social publishing automation
- Advanced billing
- Realtime collaboration

## Success Metrics

- Upload-to-first-candidate completion rate
- Candidate acceptance rate
- Editor-open rate
- Export completion rate
- Render failure rate
- Time to first export
- Repeat project creation

Use UX Flow v2, Design System v2 and ADR-UI-001 as final UI authority.
